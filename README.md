# CIDResolver

Convert Cell ID to BTS/NB/eNB ID. 

Build with Angular 12 and Material Theme

## Development

### Gitpod

Open GitPod and start develop instantly without installing anything to local computer.

[![Gitpod ready-to-code](https://img.shields.io/badge/Gitpod-ready--to--code-blue?logo=gitpod)](https://gitpod.io/#https://github.com/olkitu/CIDResolver)

### Without Gitpod

Requirements:

* NodeJS latest stable
* Angular CLI
* Yarn

Run `yarn` to install Node-modules

Run `ng serve` to start dev server. Navigate to `http://localhost:4200`. This app will automatically reload if you change any of the source file in `app` directory.
